# TCG Radar Source Map

Current MVP adapters:

- `youtube`: creator momentum and mention spikes
- `reddit`: collector sentiment and early forum confirmation
- `ebay`: listing depletion, sold count changes, sealed dislocations
- `tcgplayer`: sold-price and depth checks
- `psa`: population pressure and scarcity checks

This MVP ships with deterministic sample adapters so the scoring and UI can be exercised without live credentials.
